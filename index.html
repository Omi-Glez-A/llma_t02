Crear un nuevo repositorio en Github

Para crear un repositorio público en GitHub, sigue estos pasos:

  1. Inicia sesión en tu cuenta de GitHub o crea una nueva si aún no tienes una. Luego, haz clic en el botón "+" en la esquina superior derecha de tu página de inicio y selecciona "Nuevo repositorio".
  2. Completa los detalles de tu repositorio, incluyendo un nombre, una descripción opcional, la visibilidad (asegúrate de seleccionar "Público" para que sea visible para todos) y otras configuraciones según tus preferencias. Después, haz clic en "Crear repositorio". ¡Listo! Ahora tienes un repositorio público en GitHub donde puedes almacenar y compartir tu código con la comunidad de desarrollo.

Recuerda ser consciente de la información que compartes en repositorios públicos, ya que todo lo que subas será accesible públicamente en la plataforma.

Enlazar el repositorio en Render

Podemos crear una página estática en Render utilizando un repositorio de GitHub como el que hemos creado anteriormente siguiendo las instrucciones a continuación:

  1. Sube tu código estático a tu repositorio de GitHub: Asegúrate de que tu proyecto web esté en tu repositorio de GitHub. Esto puede ser un sitio web estático desarrollado en HTML, CSS, JavaScript u otra tecnología web. Puedes organizar los archivos en una rama "main" o "master" para que Render pueda acceder a ellos.
  2. Inicia sesión en Render: Ve a la plataforma de Render y regístrate o inicia sesión en tu cuenta.
  3. Crea un nuevo servicio en Render: En el panel de Render, selecciona "Nuevo Servicio" y elige "GitHub". Render te pedirá acceso a tu cuenta de GitHub. Concede los permisos necesarios y busca el repositorio que deseas usar para tu página estática.
  4. Configura tu servicio: Render te permitirá configurar las opciones de implementación, como la rama a desplegar y el comando de inicio (generalmente, para sitios estáticos, no se necesita un comando de inicio). Luego, haz clic en "Siguiente" y configura las variables de entorno o ajustes adicionales según sea necesario.
  5. Implementa tu página estática: Una vez configurado, Render implementará automáticamente tu proyecto desde GitHub. Espera a que el despliegue se complete.
  6. Obtén la URL de tu página: Una vez que la implementación esté completa, Render proporcionará una URL pública para tu página estática. Puedes acceder a tu sitio web desde esta URL y compartirlo con otros.

Con estos pasos, habrás creado una página estática en Render utilizando tu repositorio de GitHub como fuente de código, lo que te permitirá alojar y publicar tu sitio web de forma sencilla y gratuita.
